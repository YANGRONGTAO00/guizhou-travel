# Каталог достопримечательностей Гуйчжоу на Django

Это учебный проект Django: веб-приложение-каталог достопримечательностей провинции Гуйчжоу.

## Реализованные страницы

- Главная страница: `/`
- Список достопримечательностей: `/spots/`
- Детальная страница: `/spots/<id>/`
- Форма добавления новой записи: `/spots/add/`
- Страница подтверждения удаления: `/spots/<id>/delete/`

## Требования задания

Проект содержит несколько моделей и все три типа связей:

- `Region` → `ScenicSpot`: один-ко-многим
- `ScenicSpot` → `SpotProfile`: один-к-одному
- `ScenicSpot` ↔ `Feature`: многие-ко-многим

Также используются разные типы полей: текст, числа, даты, логическое поле и перечисление.

## Запуск

```bash
python3 manage.py migrate
python3 manage.py seed_guizhou
python3 manage.py runserver
```

Откройте в браузере:

```text
http://127.0.0.1:8000/
```

## Демонстрационные данные

Команда `seed_guizhou` создаёт русскоязычные записи о достопримечательностях Гуйчжоу, например:

- Водопад Хуангошу
- Малый Семь Арочный мост Либо
- Мяоская деревня Сицзян Цяньху
- Древний городок Цинъянь
- Туристическая зона Чишуй Данься
